# Trippin AI Public Site Export

This folder is a safe public export package for Trippin AI legal pages.

It is intended to be copied to a public GitHub Pages repository, Cloudflare Pages project, simple static website, or another public docs host.

This folder should contain only public-safe static pages. Do not add bot source code, environment files, logs, private IDs, user messages, generated reports, generated media, or internal-only docs.

## Included Pages

- `index.md`
- `legal/index.md`
- `legal/terms.md`
- `legal/privacy.md`
